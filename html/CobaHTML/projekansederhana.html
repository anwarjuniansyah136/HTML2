Tentang Projekan HTML

Projekan HTML adalah proyek sederhana yang dibuat untuk menggambarkan struktur dasar sebuah halaman web menggunakan HTML dan CSS. Halaman ini memiliki header dengan judul yang ditampilkan di tengah, dan bagian utama dengan beberapa paragraf teks.

Projekan ini juga menunjukkan penggunaan CSS untuk mengatur tata letak dan gaya elemen-elemen HTML. Misalnya, tata letak halaman menggunakan CSS Grid, dan tombol memiliki efek hover dengan perubahan warna latar belakang.

Anda dapat menggunakan projekan ini sebagai dasar untuk mempelajari dan bereksperimen dengan HTML dan CSS. Selamat berprojekan!
